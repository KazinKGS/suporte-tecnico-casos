# Caso 01 - Usuário sem acesso ao sistema

## Problema
Usuário não conseguia acessar o sistema corporativo.

## Diagnóstico
Verificado que a conta estava bloqueada no Active Directory.

## Solução
- Acesso ao AD
- Desbloqueio da conta
- Reset de senha

## Resultado
Usuário voltou a acessar normalmente.

## Observação
Problema comum causado por múltiplas tentativas de login incorretas.
